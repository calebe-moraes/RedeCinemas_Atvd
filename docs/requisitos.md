# Requisitos Funcionais

RF01 - Cadastrar cinemas
RF02 - Cadastrar filmes
RF03 - Criar sessões
RF04 - Registrar público das sessões
RF05 - Consultar filmes em cartaz
RF06 - Consultar público por cinema

# Regras de Negócio

RN01 - O público não pode ultrapassar a capacidade do cinema.
RN02 - Uma sessão deve possuir horário válido.
RN03 - Um filme pode estar em vários cinemas.
RN04 - Toda sessão deve estar vinculada a um filme.
