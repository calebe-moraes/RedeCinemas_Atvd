# Rede de Cinemas - Engenharia de Software

Projeto desenvolvido para a disciplina de Engenharia de Software.

## Objetivo
Desenvolver um sistema para gerenciamento de uma rede de cinemas utilizando:

- UML
- MVC
- Service
- Repository
- SQLite

## Funcionalidades
- Cadastro de filmes
- Cadastro de cinemas
- Controle de sessões
- Registro de público
- Relatórios

## Tecnologias
- Python
- SQLite
- GitHub
